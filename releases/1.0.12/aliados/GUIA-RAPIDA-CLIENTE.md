# Guia rapida del cliente - Fundacion Aliados

## Sistema

Sistema Comercial de Costeo para Bionegocios

## Version entregada

1.0.12

## Para instalar

1. Abrir el archivo `Sistema Comercial de Costeo para Bionegocios_1.0.12_x64-setup.exe`.
2. Completar la instalacion.
3. Abrir la app desde el acceso directo o desde el menu de Windows.

## Primer uso recomendado

1. Completar los datos de Inicio.
2. Registrar costos.
3. Definir transformacion y salidas si aplica.
4. Registrar comparables de mercado si existen.
5. Revisar Resultados.
6. Generar el PDF del informe.

## Guardar y recuperar trabajos

- `Guardar archivo`: guarda el costeo actual como archivo JSON.
- `Cargar archivo`: recupera un trabajo anterior para seguir editandolo.

Recomendacion:

Crear una carpeta institucional para conservar todos los costeos historicos.

## Actualizaciones

Usar el boton `Verificar actualizacion`.

Si existe una version nueva:

- la app avisara la version publicada
- se podra descargar la nueva entrega del canal correspondiente

## Soporte basico

Si se presenta una novedad, enviar:

- captura del error
- archivo JSON del costeo
- nombre del producto o servicio
- descripcion corta de lo que se estaba haciendo

## Observacion

Esta entrega es una edicion operativa para uso del cliente. Los cambios estructurales del producto se realizan desde la matriz administrada por Jungle Lab S.A.S.
