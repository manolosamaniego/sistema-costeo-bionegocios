# Clientes

Esta carpeta organiza la personalización y seguimiento por cliente.

## Estructura sugerida

- `_plantilla/`
  Base para un cliente nuevo.
- `aliados/`
  Cliente inicial y referencia de operación.

## Qué guardar aquí

- branding base del cliente
- notas de alcance
- observaciones de soporte
- historial de entregas

No guardar aquí respaldos operativos del cliente si contienen información sensible sin autorización.
