# Guia de entrega comercial - version 1.0.12

## Producto

Sistema Comercial de Costeo para Bionegocios

## Cliente inicial

Fundacion Aliados

## Objetivo de esta entrega

Entregar una version operativa, instalable y lista para uso institucional, con guardado de archivos, carga de trabajos anteriores, informe PDF e identificacion visual del cliente.

## Paquete principal

Ruta del paquete:

`releases/1.0.12/aliados/paquete`

Contenido esperado:

- instalador `setup.exe`
- archivo de branding del cliente
- archivo de licencia del cliente

## Instalador recomendado

Usar este archivo para la instalacion del cliente:

`Sistema Comercial de Costeo para Bionegocios_1.0.12_x64-setup.exe`

## Capacidades incluidas en esta entrega

- costeo de productos y servicios
- costos directos, indirectos, comercializacion y equipos
- transformacion, salidas, merma y asignacion de costo comun
- comparacion con productos similares
- resultados ejecutivos
- generacion de informe PDF
- guardado y recuperacion de archivos JSON
- verificacion de actualizaciones

## Validacion comercial minima

Antes de entregar al cliente, validar:

1. La app abre sin error.
2. Permite guardar un costeo.
3. Permite recuperar el mismo costeo.
4. Genera el PDF del informe.
5. El branding de Aliados aparece correctamente.
6. La version operativa no expone controles sensibles de matriz.

## Soporte

La correccion siempre se hace sobre la matriz del producto y luego se publica una nueva version para el cliente.

Documentos de apoyo:

- `docs/FLUJO-DE-SOPORTE.md`
- `docs/RUTA-DE-SOPORTE-REMOTO.md`
- `docs/MECANISMO-DE-ACTUALIZACION.md`
