# Release 1.0.0

## Tipo

Primera versión comercial estable de escritorio.

## Incluye

- aplicación Tauri para Windows
- guardado y carga de costeos
- exportación e importación de branding
- modo administrador para identidad institucional
- impresión de informe ejecutivo

## Instaladores esperados

- NSIS `.exe`
- MSI `.msi`

## Validación mínima

- instalar en equipo limpio
- abrir la app
- guardar un JSON
- cargar un JSON
- exportar branding
- cargar branding
- imprimir informe
