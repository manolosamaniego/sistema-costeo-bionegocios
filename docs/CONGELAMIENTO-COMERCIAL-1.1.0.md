# Congelamiento comercial 1.1.0

## Estado

La version `1.1.0` queda definida como base comercial congelada del Sistema Comercial de Costeo para Bionegocios.

## Alcance congelado

Incluye:
- flujo principal de costeo
- costos directos, indirectos, comercializacion y equipos
- participacion porcentual visible por rubro
- modulo de certificaciones
- transformacion, mercado, resultados e informe
- guardado y carga de archivos
- generacion de PDF
- actualizacion por manifiesto

## Regla operativa

Desde este punto:
- la version vendible no se modifica directamente
- toda mejora nueva se desarrolla en la matriz
- cada mejora debe salir en una nueva version
- la copia del cliente solo recibe instaladores compilados

## Version comercial recomendada

- version comercial congelada: `1.1.0`
- cliente inicial: `Fundacion Aliados`
- marca del producto: `Jungle Lab S.A.S.`

## Evidencia minima asociada

- licencia matriz actualizada a `1.1.0`
- licencia cliente Aliados `1.1.0`
- validacion de formulas documentada
- guias de entrega y soporte remoto ya disponibles en el proyecto
