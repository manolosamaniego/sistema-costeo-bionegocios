# Checklist laptop de viaje

## Antes de viajar

- [ ] Git instalado
- [ ] Node.js instalado
- [ ] Rust instalado
- [ ] Visual Studio Build Tools instalado
- [ ] Acceso a GitLab funcionando
- [ ] Repositorio clonado o accesible
- [ ] App compila en esa laptop
- [ ] Credenciales de Git funcionando
- [ ] Carpeta de trabajo definida

## Antes de atender soporte

- [ ] Descargar JSON del cliente
- [ ] Descargar branding si aplica
- [ ] Registrar incidencia
- [ ] Confirmar versión afectada

## Antes de entregar corrección

- [ ] Probar el mismo caso
- [ ] Compilar instalador
- [ ] Anotar versión corregida
- [ ] Guardar nota de entrega
