# Estructura de carpetas recomendada

## Carpeta matriz

Ubicación sugerida:

`C:\Proyectos\matriz-costeo-bionegocios`

Contenido:

- código fuente
- documentación
- plantillas de cliente
- historial de cambios

## Carpeta de entregas

Ubicación sugerida:

`C:\Proyectos\entregas-costeo-bionegocios`

Contenido:

- instaladores por versión
- paquetes por cliente
- notas de entrega

## Carpeta de respaldo de clientes

Ubicación sugerida:

`C:\Proyectos\respaldo-clientes-costeo`

Contenido:

- JSON recibidos por soporte
- branding exportado
- evidencia de incidencias

## Regla

No mezclar en una sola carpeta:

- código fuente
- instaladores
- archivos de soporte del cliente

Separarlo evita errores, pérdidas y confusión de versiones.
